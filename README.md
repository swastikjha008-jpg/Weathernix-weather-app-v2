# Weathernix Weather App

Weathernix is a modern weather dashboard built with React and the OpenWeather API. It shows live weather data, forecast details, nearby cities, and a favorites system inside a responsive UI with dynamic weather styling.

## Features

- Live weather search by city
- Current temperature, humidity, wind, pressure, and visibility
- 5-day weather forecast
- Nearby city weather cards
- Favorites system with local storage
- Day / night theme toggle
- Dynamic weather background and rain effect
- Responsive layout for desktop and mobile

## Tech Stack

- React
- Vite
- OpenWeather API
- Lucide React Icons
- HTML, CSS, JavaScript

## Setup

### 1. Clone the repository
```bash
git clone https://github.com/your-username/weathernix-weather-app.git
cd weathernix-weather-app
2. Install dependencies
npm install
3. Create a .env file

Add your OpenWeather API key:

VITE_OPENWEATHER_API_KEY=your_api_key_here
4. Run the app locally
npm run dev
Build for Production
npm run build
Preview Production Build
npm run preview
